## Full Description

```
Ghost Interface detects job application forms and fills them automatically using your saved profile — name, email, phone, work experience, and resume links — so you spend time on applications, not on copy-pasting.

HOW IT WORKS
1. Save your profile once in the extension popup
2. Navigate to any supported job application page
3. Click the Ghost pill that appears in the corner — Ghost Interface maps your profile to the right fields and fills them instantly

WHAT MAKES IT DIFFERENT
Every filled field gets a color-coded confidence ring so you can see how certain the extension was about each value:
• Green — exact match from your profile
• Amber — fuzzy or inferred value
• Red — open-ended question that needs your input
• Gray — field left blank (unknown type)

A Decision Tree panel shows every field, its confidence score, and the fill method used. A Submit Gate blocks the form if any field is below the confidence threshold, so you never send an incomplete application.

For open-ended questions like "Why do you want to work here?", you get three options: a pre-written template, an AI-generated answer tailored to the company and role, or a blank editor to write your own.

SUPPORTED PLATFORMS
Ghost Interface works with the most widely used applicant tracking systems. It recognizes Greenhouse application pages at boards.greenhouse.io, Workday at myworkdayjobs.com domains, Lever at jobs.lever.co, iCIMS at icims.com domains, BambooHR at bamboohr.com, Ashby at jobs.ashbyhq.com, and Loxo at app.loxo.co. The extension detects the platform automatically and adapts its field mapping to each form structure.

PRIVACY
All profile data is stored locally in your browser using chrome.storage.local. Nothing is sent to external servers. An Anthropic API key is only required if you want AI-generated answers for open-ended fields — and even then, your key never leaves your device.

REQUIREMENTS
• Chrome 114 or later
• An Anthropic API key is optional (needed only for AI open-ended answers)
• Basic form autofill works without any API key
```
